Minnesota Mountain Biking

For my foundations capstone I wanted to create an easy to use, local mountain biking application where users could go to find trail/park ideas and customize some of the information for themselves.

Tech stack: Javascript, PosgreSQL, HTML/CSS, Express/Axios

Project Features:
  • Seed database of park options
  • Add a park to the database
  • Sort/filter the database
  • Add a park to favorites
  • Remove a park from favorites
  • Add personal notes to a favorited park
  • Responsive format makes for an easy transition to be mobile-friendly
