Project information

Tech stack: JavaScript, Java(Spring Boot), HTML/CSS

Project Features:
  • User registration, login and logout
  • Personal booklist/notes based on user login
  • CRUD operations on any book
  • Add/remove sticky notes to books
  • Sorts books into read and unread sections

    • Bookworm buddy is an organizational tool for reading enthusiasts. My application is designed to help readers organize books they have read, want to read and any keep track of any personal notes and takeaways all in one place.
